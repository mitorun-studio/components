Контейнеры (ограничители контента)

Контейнер №1 .container

Классический контейнер с отдельным дивом.

Контейнер .container-fluid

Контейнер на всю ширину экрана.

Контейнер №2 .container-2

Контейнер без отдельного элемента. Имеет шаги с фикс-шириной и поведение как у бутстрапа (540px, 720px, 960px, 1140px, 1320px).

Контейнер №3 .container-3

Контейнер без отдельного элемента. Это упрощенная вариация с плавным изменения ширины.

Есть нюанс, что при padding: 0 calc(50vw - 570px); ширина контейнера не 1140px, а 1123px - из-за того что 17px отнимает на себя скроллбар. 50% решило проблему

Контейнер №4 .container-4

Самая простая и перспективная версия контейнера! Без отдельного дива и одним свойством.

Сделано с помощью функции max(). Еще возможны варианты с minmax() и clamp(), но они вероятно или хуже или вообще не подходят.

Контейнер №4-fluid .container-4-fluid

Версия контейнера №4 на всю страницу.

Контейнер №5 .container-5

Вариант контейнера от Kevin Powell и Стефани Эклз.

Сделано с помощью функции width: min().

Контейнер №6

Вариант контейнера на гридах с помощью свойства grid-template-columns.

body {
	display: grid;
	grid-template-columns: minmax(1rem, 1fr) minmax(0, 65ch) minmax(1rem, 1fr);
	grid-template-areas:
	"header header header"
	". main ."
	". footer .";
}
