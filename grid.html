Хедер + Мейн + Футер

Мейн прижимает футер к низу страницы, даже если на странице мало контента.

Header
Main
Footer
<div class="grid">
<div class="grid-element">Header</div>
<div class="grid-element">Main</div>
<div class="grid-element">Footer</div>
</div>

.grid {
display: grid;
grid-template-rows: auto 1fr auto;
height: 100%;
}

Адаптивная картинка и наложение текста

Heading

<div class="grid">
<img class="grid-img">
<p class="grid-p">Heading</p>
</div>


.grid {
display: grid;
grid-template: 'container';
place-content: center;
place-items: center;
max-height: clamp(200px, 20vh, 400px);
overflow: hidden;
}
.grid-img {
grid-area: container;
width: 100%;
max-width: 500px;
height: auto;
aspect-ratio: 16 / 9;
object-fit: cover;
}
.grid-p {
grid-area: container;
max-width: 500px;
}
