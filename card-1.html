Галерея с различными карточками

Правила создания карточек:

  • Нужно ли использовать элемент список для карточек? Для каких случаев точно нужен список, а для каких точно не нужен?
  • Если в карточке есть заголовок (обычно <h3>), то он должен идти первым в HTML.
  • Не очень правильно оборачивать всю карточку в ссылку в HTML. Если нужно всю карточку сделать кликабельной, то лучше ссылкой сделать название заголовка карточки и к ней добавить псевдоэлемент, который растянуть позиционированием на всю карточку.
  • Элементы внутри карточки хорошо располагать гридами. Тогда можно гибко задавать размеры элементов, их расположения (менять местами), расположить кнопку например справа и т. п.
  • Селекторы в карточках желательно делать такими, чтобы они не зависели от используемых элементов (h3, p, a, button...).
  • Для текстового содержимого карточек можно добавить для профилактики max-width: 60ch;, это ограничивает ширину текстовых блоков если карточка сильно растянется.
  • Для скринридеров (и поисковиков?) будет плохо, если карточки будут содержать ссылки с одинаковым текстом типа "Читать далее", "Подробнее". Например можно сделать ссылкой всю карточку, а кнопку с текстом "Подробнее" сделать чисто декоративной: <span class="button" aria-hidden="true">Подробнее</span>
